#  VISHGUARD Dataset (Public Overview)

**VISHGUARD** is a multilingual synthetic audio dataset designed for research on voice phishing (vishing) detection and persuasion-aware speech analysis.  
It supports multilingual experiments (English, French, Arabic) and includes annotations on emotional tone and persuasion strategies.

 **Main Features**
- 3,000 simulated phone calls (fraudulent and legitimate)
- Balanced multilingual distribution
- Persuasion and emotion annotations
- Ethical generation process using synthetic voices
- For research and evaluation purposes only
